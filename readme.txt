== Installation ==
Download the plugin zip file from the WaMate Confirm plugin page.
Log in to your WordPress admin panel.
Navigate to Plugins > Add New.
Click on the "Upload Plugin" button at the top of the page.
Choose the downloaded zip file and click "Install Now."
Once installed, activate the plugin through the Plugins menu in WordPress.
Usage:

After activation, navigate to WaMate > Settings.
Click on the newly added "Settings" Submenu.
Customize your order notification settings:
Enter a custom message for order notifications. You can use placeholders like [name], [orderid], [dlv], and [cost] to personalize the message with customer details and order information.
Specify the country code for WhatsApp messages.
Provide the Instance ID and Access Token for the API.
Save your changes.
You Need The Instance ID and Access Token From http://wamate.online

== Frequently Asked Questions ==

Q: Can I use this plugin with other e-commerce platforms?
A: No, this plugin is specifically designed for WooCommerce and will only work with it.

Q: Can I change the message template after I've set it up?
A: Yes, you can change the message template at any time from the 'Order Notification' tab.

Q: Can I add a country code for international customers?
A: Yes, you can add a country code for international customers from the 'Order Notification' tab.
